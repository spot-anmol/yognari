# CLAUDE.md

Context for working in this repo.

## What this is
`yognari` — a single-page marketing landing page for "Aanya", a yoga / yog nidra / breathwork guide.
Originally generated by Lovable; **Lovable is no longer used** (free to rewrite git history, remove Lovable-specific bits).

## Stack
- TanStack Start (React 19) on Vite 8, Tailwind v4, shadcn/ui components (`src/components/ui`).
- Routes in `src/routes` (`index.tsx` is the only real page; `__root.tsx` is the shell). No server functions — it's effectively static.

## Tooling
- **Use `npm`** — the toolchain is all-npm, both locally and in CI (committed `package-lock.json`; no bun).
- `npm install` → `npm run build` (build output: **`dist/client/`**, fully prerendered static HTML).
- `npm run dev` for local dev.

## Build = static prerender (important)
`vite.config.ts` sets `tanstackStart.prerender.enabled` to emit static HTML for free static hosting.
Do **not** force `nitro: { preset: "static" }` or `spa` — both break the build under this stack (Vite 8 + Nitro v3 beta): prerender 404s on `/` and nitro errors on an html SSR input. Plain prerender works.

## Deploy
- Host: **Firebase Hosting**, project ID `yognari` (free Spark plan). Config in `firebase.json` / `.firebaserc`.
- Live domain: **https://yognari.in** (apex primary; `www` redirect not yet set up). Mirror: `yognari.web.app`.
- Manual deploy: `firebase deploy --only hosting`.
- Auto-deploy: pushing to `main` triggers `.github/workflows/firebase-hosting-deploy.yml` (needs repo secret `FIREBASE_SERVICE_ACCOUNT_YOGNARI`).
